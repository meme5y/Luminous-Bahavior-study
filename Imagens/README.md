# IMAGES FOLDER

This folder contains visual documentation of the experiment.

## Required Images:
1. `experimental_setup.jpg` - Photo of the complete hardware setup
2. `circuit_diagram.png` - Schematic of the electronic circuit
3. `results_charts.png` - Key statistical visualizations
4. `participant_setup.jpg` - How participants interact with the system

## Upload Instructions:
1. Take clear photos of your setup
2. Name files descriptively
3. Keep file sizes reasonable (<2MB)
4. Ensure good lighting and focus

## For Publication:
- Use high-resolution images
- Include scale references
- Show important details clearly
